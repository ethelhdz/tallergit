git init
genera un archivo .git (no se ve)

Directorios: Working directory --> index --> head
			       add     commit

git status //compara working directory con los 2 arboles

se crea archivo .txt 
git add archivo.txt//crear archivo

git status //se muestra el nuevo archivo agregado

git reset //para no trackear (deshacer el git add)

git stash //guardar todos los cambios 
  -p vas   	guardando por pedazos
  --patch  
  pop      	regresa al ultimo cambio que guardo
  drop     	elimina los cambios
  clear    	limpiar la pila de los cambios

git commit 	todo lo de index lo para al head (historia de los archivos)
  -p 		
  -m "mensaje"	agregar un mensaje

git log 	//historial de proyecto
